# SETUP

A script that automatically sets up development environment.
Currently supports Windows, Mac(darwin), Linux operating systems

## WHAT IT CAN DO NOW
Scans specified directory for the tool e.g "python". Installs it if it doesnt exists.

## TEST RUN

```bash
# navigate to the script's directory. Run
python setup_dev.py --env=[your tool e.g python] --dir=[your_project_path]
```
